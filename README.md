 # NotReallyNetworking

 A skill-based platform where students can earn opportunities without relying on connectoins.

 ## Problem 
 Many students struggle to get internships due to lack of connections, not of skills.

 ## Solution
 This platform allows students to prove their skills by solving challenges instead of relying on networking.

 ## Tech stack
 - Node.js
 - Express.js
 - MongoDB
 - Mongoose

 ## How to run

 ### Start MongoDB
 Run:
 mongod

 ### Start Server
 cd server
 npm start
